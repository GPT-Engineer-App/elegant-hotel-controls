# elegant-hotel-controls

A business dashboard showing controls suitable for a hotel guest, include options to contact housekeeping, contact lobby, contact restaurant, contact spa, book an activity, look at the events schedule and ask for transportation. It should look elegant and professional.

## Setup

```sh
git clone https://github.com/GPT-Engineer-App/elegant-hotel-controls.git
cd elegant-hotel-controls
npm i
```

```sh
npm run dev
```

This will run a dev server with auto reloading and an instant preview.
